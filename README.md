# Recepta — ИИ-администратор для локального бизнеса

Telegram-бот, который **отвечает клиентам 24/7 и доводит до записи** — как живой
администратор, только не спит и не теряет заявки, что приходят ночью или в
выходной. Демо собрано на вымышленном салоне красоты **ВЕЛЬВЕТ** (тот же бренд,
что и в [демо-лендинге](https://velvet-salon-landing.vercel.app)).

> Продаётся не «бот», а результат: **перестаёте терять клиентов, которые пишут не
> в рабочее время.** База знаний одна на бизнес — под нового клиента меняется
> только `data/salon.json`.

## Что умеет

- **Отвечает на вопросы** про услуги, цены, часы, адрес, оплату, парковку.
- **Записывает** прямо в чате (услуга → имя → телефон → удобное время) и шлёт
  заявку администратору + пишет в базу.
- **Быстрые кнопки** для частых запросов.
- **Два режима ответа:**
  - **Без ключа** — встроенный FAQ-движок (ключевые слова + нечёткое совпадение).
    Бесплатно, работает офлайн, демо никогда не падает.
  - **С ключом Anthropic** — Claude отвечает живым языком, но **строго по базе
    знаний** салона (никаких выдуманных цен и адресов). При любом сбое —
    автоматический откат на встроенный движок.

## Стек

Python 3.12 · aiogram 3 (FSM) · SQLite · Anthropic SDK (опционально) ·
модель по умолчанию `claude-haiku-4-5` (дёшево и быстро под поток ресепшена;
можно поднять до `claude-opus-4-8`).

## Быстрый старт

```bash
python -m venv .venv && . .venv/Scripts/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env        # впишите BOT_TOKEN от @BotFather
python -m app
```

Ключ Anthropic **не обязателен** — без него бот сразу работает на FAQ-движке.
Чтобы включить живые ответы Claude, добавьте `ANTHROPIC_API_KEY` в `.env`.

### Посмотреть без токена

```bash
python demo.py
```

Прогонит типичные вопросы клиента через встроенный движок — удобно для
скриншотов в портфолио.

## Как переиспользовать под другой бизнес

Весь контент — в `data/salon.json`: реквизиты, услуги с ценами и синонимами,
график, FAQ, тон общения. Меняете JSON — получаете администратора под барбершоп,
клинику, автосервис и т.д. Код трогать не нужно.

## Коммерциализация (движок #1)

- **Setup** 15–40к ₽ (настройка под бизнес: база знаний, тон, интеграции).
- **Абонентка** 3–8к ₽/мес (поддержка, правки прайса, хостинг).
- Первому клиенту — setup почти бесплатно за отзыв и кейс.

## Структура

```
app/
  config.py      # переменные окружения
  knowledge.py   # база знаний + keyless FAQ-движок
  ai.py          # адаптер Claude (grounded, с откатом)
  booking.py     # запись в SQLite
  handlers.py    # хендлеры + FSM записи
  bot.py         # сборка и запуск (long polling)
data/salon.json  # ← контент бизнеса (единственное, что меняется под клиента)
demo.py          # оффлайн-демо без токена
```

## Заметки по деплою

- Для постоянной работы боту нужен всегда включённый процесс: VPS,
  `systemd`/`pm2`, или хостинг с фоновыми воркерами. Long polling — вебхук не
  требуется.
- `data/bookings.db` создаётся автоматически и не коммитится (в `.gitignore`).
- Секреты только в `.env` (тоже в `.gitignore`).
